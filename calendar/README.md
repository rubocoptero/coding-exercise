# Calendar

## Minimum completed project

The index.html asked for the exercise is located at `dist/index.html`


## Rest of the project

### Requirements
 - Node
 - `$ npm install -g gulp bower`

### Setup
```
$ npm install
$ bower install
```

### Possible commands
```
$ gulp serve
```
```
$ gulp build
```
```
$ gulp test
```
```
$ gulp protractor
```
